# AspectsofPower System

![Foundry v13](https://img.shields.io/badge/foundry-v13-green)

This is a project to implement a custom TTRPG made by yours truly. It originally started using the base of Boilerplate and Custom System Maker, both of which assist in creating your own TTRPG within the FoundryVTT environment. They are exceptional starting points even for a non-dev like me. My original implementation was exclusively written based on their code and I cannot overstate how much those devs have helped me in turning this into reality.

Aspects of Power is an attempt to make a progression digital TTRPG. A hybrid of a video game and a TTRPG, digital TTRPGs require computers to perform calculations but the rest is easily handled by a GM. This is still a WIP and is being heavily iterated upon.
